# NCEA Spanish Practice App

## 快速开始

### 1. 安装依赖
\`\`\`bash
npm install
\`\`\`

### 2. 配置环境变量
复制 `.env.example` 为 `.env` 并填入你的Supabase凭据：
\`\`\`bash
cp .env.example .env
\`\`\`

### 3. 本地运行
\`\`\`bash
npm run dev
\`\`\`

### 4. 构建生产版本
\`\`\`bash
npm run build
\`\`\`

## 部署到Vercel

1. 将代码推送到GitHub
2. 在Vercel导入项目
3. 添加环境变量
4. 部署！

详细指南请查看 `deployment-guide.md`
